What is the Code Reviewer Agent?

The Code Reviewer Agent is an AI-powered code review specialist that analyzes your codebase for security vulnerabilities, performance issues, best practices violations, and code quality concerns. It provides detailed feedback and actionable recommendations to improve your code.

Installation

Install the Code Reviewer Agent using the Claude Code Templates CLI:

# Install code reviewer agent
npx claude-code-templates@latest --agent code-reviewer

# Install with security-focused agents
npx claude-code-templates@latest --agent code-reviewer --agent security-auditor

Security Vulnerability Detection

Scan your codebase for common security vulnerabilities:

# Start Claude Code
claude-code

# Request security review
> Review this codebase for security vulnerabilities, focusing on SQL injection, XSS, CSRF, authentication issues, and insecure dependencies

Security checks performed:

  • SQL injection vulnerabilities in database queries
  • Cross-Site Scripting (XSS) in user input handling
  • Cross-Site Request Forgery (CSRF) protection
  • Insecure authentication and authorization
  • Hardcoded secrets and API keys
  • Vulnerable dependencies (npm audit, pip check)
  • Insecure file uploads and path traversal
  • Missing input validation and sanitization

Identify performance bottlenecks and optimization opportunities:

# Request performance review
> Analyze the performance of this application. Look for N+1 queries, inefficient loops, memory leaks, and unnecessary re-renders. Suggest optimizations

Performance areas analyzed:

Area Issues Detected
Database Queries N+1 queries, missing indexes, inefficient joins, lack of query optimization
Frontend Performance Unnecessary re-renders, large bundle sizes, unoptimized images, blocking scripts
Algorithm Efficiency Inefficient loops, nested iterations, unnecessary computations
Memory Management Memory leaks, large object allocations, circular references
